About TOON Format

TOON (Token-Oriented Object Notation) is a compact, human-readable encoding of the JSON data model, specifically optimized for Large Language Model (LLM) prompts. It combines YAML-style indentation for nested objects with CSV-style tabular layout for uniform arrays, resulting in significant token savings. This format is ideal for developers working with AI models, reducing API costs and improving prompt efficiency by 30-60% compared to standard JSON.

Compact
